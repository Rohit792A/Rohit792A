### 🔲 Smart Attendance System via QR Code
> **Python · OpenCV · MyQR · NumPy · pybase64**

Built a fully automated attendance recording system using QR code scanning — replacing manual sign-in sheets entirely. The system generates unique QR codes per student, scans them in real-time using OpenCV, and logs attendance automatically. Presented as a complete working prototype with a report and slide deck.

📌 *Key outcome: Eliminated manual attendance effort, reduced proxy attendance risk.*
